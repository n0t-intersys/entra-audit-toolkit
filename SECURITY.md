# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this toolkit, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Contact: **contact@intersystechnology.com**

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested remediation (if any)

I aim to acknowledge reports within 48 hours and provide a resolution timeline within 7 days.

## Scope

This policy covers the PowerShell scripts in this repository. It does not cover vulnerabilities in Microsoft Graph API, Microsoft Entra ID, or the `Microsoft.Graph` PowerShell SDK — report those directly to Microsoft via [msrc.microsoft.com](https://msrc.microsoft.com).

## Supported Versions

Only the latest version on the `main` branch is actively maintained.
